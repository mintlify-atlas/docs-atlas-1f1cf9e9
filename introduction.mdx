# Python Arcade Suite

Python Arcade Suite is a **SaaS web application** featuring a collection of interactive classic arcade games and logic puzzles, built entirely in Python and powered by Streamlit. Experience the nostalgia of classic games with a modern, responsive web interface.

## Why Python Arcade Suite?

This project goes beyond simple game scripting by implementing professional-grade web application features:

### Session State Management

Unlike standard Python scripts, Streamlit re-runs code on every user interaction. Python Arcade Suite effectively manages state to persist:
- Game scores across rounds
- Card hands in Blackjack
- Guessed letters in Hangman
- Match history in Rock, Paper, Scissors

<Info>
  All game state is managed using `st.session_state`, ensuring smooth gameplay without losing progress between interactions.
</Info>

### Modular Architecture

The codebase follows a clean, modular design pattern:

```plaintext
streamlit_app.py          # Main entry point and game selector
├── blackjack/
│   └── streamlit_game.py # Blackjack game logic
├── Hangman/
│   └── streamlit_game.py # Hangman game logic
└── Rock_paper_or_scissors/
    └── streamlit_game.py # RPS game logic
```

Each game is self-contained with its own `app()` function, making the code maintainable and easy to extend.

### Responsive User Interface

The application features:
- **Interactive widgets**: Buttons, text inputs, and selectors for seamless gameplay
- **Real-time updates**: Instant visual feedback using Streamlit's rerun mechanism
- **Visual feedback**: Success/error messages, balloons on wins, and ASCII art for Hangman
- **Sidebar navigation**: Easy switching between games without losing state
